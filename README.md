## 기능
- 할 일과 예상시간을 작성하고 추가하면 리스트에 등록
- 아이템의 총 갯수, 총 예상시간, 예상시간 미등록 갯수 출력
- 등록된 아이템은 각각의 체크리스트를 작성 가능
- 체크리스트는 class 바인딩으로 완료와 진행중 구분
- 할 일의 진행상태를 '진행중'과 '완료'로 구분
- 진행 상태를 선택하면 선택한 진행상태에 맞게 아이템이 추가됨
- 검색 컴포넌트에서 아이템을 검색 가능
